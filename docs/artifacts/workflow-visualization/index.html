Video Pipeline System

Flujo completo del sistema de generación automática de videos para Instagram Reels

Vista General del Flujo

Flujo Paso a Paso (13 Pasos)

0

Selección de Proyecto

Web UI

El usuario selecciona un proyecto existente o crea uno nuevo. El proyecto define el contextPath que usará OpenCode.

Promo Mayo
5 videos
Testimonios
12 videos
+ Nuevo
1

Usuario Submit Prompt

Web UI

El usuario escribe un prompt dentro del proyecto seleccionado.

POST /api/projects/:projectId/videos
{ "prompt": "Video promocional de membresías" }
2

API Crea Video Record

SQLite

La API crea un registro en SQLite con status = draft.

status: draft progress: 0%
3

Spawn OpenCode ACP Session

OpenCode

La API inicia OpenCode en modo ACP y crea una sesión JSON-RPC sobre stdin/stdout.

$ opencode acp --cwd ./content/promo-mayo
4

OpenCode Analiza Contexto del Proyecto

AI Agent

OpenCode lee los archivos del contextPath del proyecto (markdown, imágenes, etc.) y genera una idea de video contextualizada.

Lee ./content/promo-mayo/
Analiza tema
Genera idea
5

Idea POSTed a la API

Webhook

OpenCode envía la idea generada de vuelta a la API mediante POST. El status cambia a idea_ready.

POST /api/videos/:id/idea
{ "title": "Promo Verano", "scenes": [...] }
6

Usuario Aprueba Idea

UI Action

El usuario revisa la idea en la UI y aprueba para continuar (o puede regenerar).

Aprobar Regenerar
7

Creación de Scenes en DB

Database

La API crea registros para cada scene (clip) en SQLite. Status = generating_clips.

3-5
scenes
Cada scene tiene:
• prompt para MiniMax
• duración (6-10s)
• descripción
8

MiniMax Genera Clips

AI Video

MiniMax Hailuo AI genera cada clip en paralelo. Polling cada 5s hasta completar.

9

Download de Clips

Storage

Cuando MiniMax completa cada clip, se descarga al filesystem local. Status = clips_ready.

./storage/clips/scene-1-abc123.mp4
./storage/clips/scene-2-def456.mp4
./storage/clips/scene-3-ghi789.mp4
10

Preview en Remotion Player

Preview

El usuario puede previsualizar el video con todos los clips combinados usando el Remotion Player.

• Player embebido
• Controles de playback
• 1080x1920 (Reels)
11

Render Final

Remotion

Usuario solicita export. Remotion renderiza el video final con todos los clips y overlays.

status: composing 30fps · H.264
12

Video Completo - Download

Completed

El video final está listo para descargar. Status = completed.

promo-mayo-final.mp4
1080x1920 · 24s · 12MB

Flujo de Datos

Input

  • Prompt del usuario
  • Contexto (markdown, imgs)
  • Configuración (YAML)

Procesamiento

  • OpenCode genera idea
  • MiniMax genera clips
  • Remotion compone

Output

  • Video MP4 (Reels)
  • Metadata en SQLite
  • Clips individuales
