Arquitectura Agentica

Estrategia de capas para generación de video

Multi-Agent Feedback Loop

Vision Estratégica

Cada capa (layer) es un agente especializado con su propia estrategia de prompting, consulta a IA y método de integración. Los agentes no son pasos secuenciales rígidos, sino nodos en un grafo de decisión que pueden iterar, bifurcar y converger basándose en feedback del usuario o de otros agentes.

Las 4 Capas Agenticas

1

Concept Layer

Agente Director Creativo

OpenCode

Estrategia de Prompting

  • Contexto jerárquico: Project → Video → brief del usuario
  • Output estructurado: JSON con título, descripción, arco narrativo
  • Few-shot: Ejemplos de videos exitosos del mismo proyecto

Decisiones del Agente

Número de escenas Duración total Arco emocional CTA strategy

→ Entrega a Layer 2

{ concept, scenes: [{description, emotion, duration}] }
2

Prompt Layer

Agente Prompt Engineer

OpenCode

Estrategia de Prompting

  • Meta-prompting: "Eres un experto en MiniMax prompts..."
  • Video-native: Términos de cámara, movimiento, luz
  • Negative prompts: Qué evitar en la generación

Decisiones del Agente

Estilo visual Movimiento de cámara Color grading Prompt negativo

→ Entrega a Layer 3

{ minimax_prompt, negative_prompt, style_params }
3

Generation Layer

Agente Ejecutor (MiniMax)

MiniMax API

Estrategia de Generación

  • Async polling: Cola de generación con retry
  • Quality gate: Validación del output (duración, calidad)
  • Auto-iterate: Si falla, consulta a Layer 2 para ajustar prompt

Decisiones Autónomas

Retry con variación Split escena larga Upscale si es fuzzy

→ Entrega a Layer 4

{ clip_path, metadata: {duration, resolution, quality_score} }
4

Composition Layer

Agente Post-Productor

Remotion FFmpeg

Estrategia de Composición

  • Análisis de clips: Detectar cortes naturales, beats
  • Overlay generation: Textos, logos, CTAs animados
  • Transiciones: Match cuts, transiciones fluidas

Decisiones del Agente

Posición texto Timing de CTAs Color correction Audio sync

→ Output Final

{ final_video.mp4, composition_manifest.json }

class ConceptAgent {
  async generateConcept(projectId: string, brief: string): Promise<VideoConcept> {
    const project = await this.projectService.getById(projectId);

    // Estrategia: Contexto jerárquico + few-shot examples
    const systemPrompt = await this.buildContextPrompt(project.contextId);

    const conceptSchema = z.object({
      title: z.string(),
      description: z.string(),
      scenes: z.array(z.object({
        sequence: z.number(),
        description: z.string(),
        emotion: z.enum(['excitement', 'urgency', 'motivation']),
        duration: z.number().min(3).max(10),
        keyMessage: z.string(),
      })),
      cta: z.object({
        text: z.string(),
        timing: z.number(), // en qué segundo aparece
      }),
    });

    const response = await this.openCodeSDK.sendPromptStructured(
      `${systemPrompt}\n\nBrief del usuario: ${brief}\n\nGenera un concepto de video para Instagram Reels.`,
      conceptSchema,
      { retryCount: 2 }
    );

    // Decisiones autónomas del agente
    const concept = response.data;
    const totalDuration = concept.scenes.reduce((sum, s) => sum + s.duration, 0);

    if (totalDuration > 60) {
      // Agente auto-ajusta: sugiere split en múltiples videos
      concept.splitRecommendation = this.calculateOptimalSplit(concept.scenes);
    }

    return concept;
  }
}
Layer 2: PromptEngineerAgent
prompt-agent.ts 
class PromptEngineerAgent {
  async craftMiniMaxPrompt(scene: Scene, feedback?: string): Promise<MiniMaxPrompt> {

    // Estrategia: Meta-prompting con contexto técnico de MiniMax
    const metaPrompt = `Eres un experto en prompts para MiniMax Hailuo AI.

Directrices técnicas:
- MiniMax funciona mejor con prompts en inglés
- Incluir términos de cámara: "panning shot", "static camera", "drone view"
- Describir movimiento: "slow motion", "dynamic action", "smooth transition"
- Especificar iluminación: "golden hour", "studio lighting", "dramatic shadows"

Contexto de la escena:
- Descripción: ${scene.description}
- Emoción: ${scene.emotion}
- Duración: ${scene.duration}s
${feedback ? `- Feedback de iteración anterior: ${feedback}` : ''}`;

    const promptSchema = z.object({
      prompt: z.string().max(500), // MiniMax limit
      negativePrompt: z.string(),
      styleModifiers: z.array(z.string()),
      cameraDirection: z.enum(['static', 'pan', 'zoom', 'tracking', 'drone']),
      lighting: z.enum(['natural', 'studio', 'dramatic', 'golden_hour']),
    });

    const response = await this.openCodeSDK.sendPromptStructured(metaPrompt, promptSchema);

    return {
      prompt: this.optimizeForMiniMax(response.data.prompt),
      negativePrompt: response.data.negativePrompt,
      parameters: {
        duration: scene.duration as 6 | 8 | 10,
        resolution: '1080p',
      },
    };
  }

  // Auto-iteración: Si el video previo falló, ajusta el prompt
  async regenerateWithFix(failedAttempt: SceneAttempt, error: string): Promise<MiniMaxPrompt> {
    const fixStrategy = this.analyzeError(error);

    const feedback = `El video anterior falló con: "${error}".
Aplica esta estrategia de fix: ${fixStrategy}`;

    return this.craftMiniMaxPrompt(failedAttempt.scene, feedback);
  }

  private analyzeError(error: string): string {
    if (error.includes('blurry')) return 'Agrega "sharp focus, 4k, highly detailed"';
    if (error.includes('text')) return 'Evita texto en la imagen; usa "no text, no watermark"';
    if (error.includes('face')) return 'Mejora calidad facial: "perfect face, detailed features"';
    return 'Simplifica el prompt, reduce elementos complejos';
  }
}
Layer 3: GenerationAgent
generation-agent.ts 
class GenerationAgent {
  private promptAgent = new PromptEngineerAgent();
  private miniMaxService = new MiniMaxService();

  async generateWithAutoRetry(scene: Scene, maxRetries = 2): Promise<GenerationResult> {
    let attempt = 0;
    let lastError: string | null = null;

    while (attempt <= maxRetries) {
      try {
        // Obtener o regenerar prompt
        const miniMaxPrompt = attempt === 0
          ? await this.promptAgent.craftMiniMaxPrompt(scene)
          : await this.promptAgent.regenerateWithFix({ scene, attempt }, lastError!);

        // Crear tarea en MiniMax
        const task = await this.miniMaxService.createVideoGeneration({
          prompt: miniMaxPrompt.prompt,
          duration: miniMaxPrompt.parameters.duration,
          resolution: miniMaxPrompt.parameters.resolution,
        });

        // Polling con quality gate
        const result = await this.pollWithQualityGate(task.taskId, scene);

        if (result.quality === 'good') {
          return { success: true, clipPath: result.path, metadata: result.metadata };
        }

        // Quality gate falló, trigger auto-retry
        lastError = `Quality issue: ${result.qualityIssues.join(', ')}`;
        attempt++;

      } catch (error) {
        lastError = error instanceof Error ? error.message : 'Unknown error';
        attempt++;
      }
    }

    // Todos los retries fallaron → Escalar a Layer 2 con feedback completo
    return this.escalateToLayer2(scene, lastError!);
  }

  private async pollWithQualityGate(
    taskId: string,
    scene: Scene
  ): Promise<QualityGateResult> {
    const clipPath = await this.miniMaxService.waitForCompletion(taskId);

    // Quality gate automático
    const analysis = await this.analyzeVideoQuality(clipPath);

    const issues: string[] = [];
    if (analysis.blurScore < 0.7) issues.push('blurry');
    if (analysis.duration < scene.duration - 1) issues.push('truncated');
    if (analysis.hasWatermark) issues.push('watermark detected');

    return {
      path: clipPath,
      quality: issues.length === 0 ? 'good' : 'poor',
      qualityIssues: issues,
      metadata: analysis,
    };
  }

  private async escalateToLayer2(scene: Scene, error: string): Promise<never> {
    throw new GenerationEscalationError(
      'Generation failed after retries',
      { scene, error, requiresUserReview: true }
    );
  }
}

class CompositionAgent {
  async composeFinalVideo(videoId: string): Promise<CompositionResult> {
    const video = await this.videoService.getById(videoId);
    const scenes = await this.sceneService.getByVideoId(videoId);

    // Análisis de beats y cortes naturales
    const beatAnalysis = await this.analyzeSceneBeats(scenes);

    // Generar overlays con Remotion (React code)
    const compositionCode = await this.generateRemotionComposition({
      video,
      scenes,
      overlays: this.calculateOverlays(video, scenes),
      transitions: this.calculateTransitions(beatAnalysis),
    });

    // Render con Remotion
    const remotionBundle = await this.bundleComposition(compositionCode);

    // Concatenar clips con FFmpeg
    const finalVideo = await this.ffmpegConcat({
      scenes: scenes.map(s => ({
        clip: s.clipPath,
        overlay: s.overlayPath,
        duration: s.duration,
        transition: s.transition,
      })),
      output: `videos/${videoId}/final.mp4`,
    });

    return {
      finalPath: finalVideo,
      manifest: {
        scenes: scenes.length,
        totalDuration: scenes.reduce((sum, s) => sum + s.duration, 0),
        overlays: compositionCode.overlays.length,
      },
    };
  }

  private calculateOverlays(video: Video, scenes: Scene[]): Overlay[] {
    const overlays: Overlay[] = [];

    // Logo persistente en corner
    overlays.push({
      type: 'logo',
      asset: `projects/${video.projectId}/assets/logo.png`,
      position: { x: 20, y: 20 },
      scale: 0.15,
      duration: 'full',
    });

    // CTAs por escena
    scenes.forEach((scene, index) => {
      if (scene.context?.textOverlay) {
        overlays.push({
          type: 'text',
          text: scene.context.textOverlay.headline,
          position: scene.context.textOverlay.position,
          animation: index === 0 ? 'fadeIn' : 'slideUp',
          timing: { start: 0.5, duration: scene.duration - 1 },
        });
      }
    });

    return overlays;
  }

  private async generateRemotionComposition(config: CompositionConfig): Promise<string> {
    // Agente genera código React/Remotion
    const prompt = `Genera un componente Remotion que:
- Renderice en 1080x1920 (9:16)
- Aplique estos overlays: ${JSON.stringify(config.overlays)}
- Use transiciones: ${JSON.stringify(config.transitions)}
- Duración total: ${config.scenes.reduce((s, sc) => s + sc.duration, 0)}s

Responde SOLO con el código TypeScript válido.`;

    const code = await this.openCodeSDK.sendPrompt(prompt);
    return this.validateAndFormatRemotionCode(code.content);
  }
}
Orchestrator: Agente Coordinador
orchestrator.ts 
class VideoOrchestrator {
  private agents = {
    concept: new ConceptAgent(),
    prompt: new PromptEngineerAgent(),
    generation: new GenerationAgent(),
    composition: new CompositionAgent(),
  };

  async processVideoRequest(request: VideoRequest): Promise<VideoResult> {
    const state: WorkflowState = {
      videoId: generateId(),
      status: 'concept_pending',
      layers: {},
    };

    // Layer 1: Concepto
    const concept = await this.agents.concept.generateConcept(
      request.projectId,
      request.brief
    );
    state.layers.concept = { status: 'completed', data: concept };

    // Approval gate: Usuario revisa y aprueba concepto
    await this.waitForUserApproval('concept', state.videoId);

    // Layers 2-3 por cada escena (paralelo con manejo de errores)
    const scenePromises = concept.scenes.map(async (sceneConfig, index) => {
      const scene = await this.createScene(state.videoId, index + 1, sceneConfig);

      try {
        // Layer 2: Prompt
        const prompt = await this.agents.prompt.craftMiniMaxPrompt(scene);
        scene.minimaxPrompt = prompt;

        // Layer 3: Generación (con auto-retry)
        const result = await this.agents.generation.generateWithAutoRetry(scene);

        scene.status = 'completed';
        scene.clipPath = result.clipPath;

        return scene;
      } catch (error) {
        if (error instanceof GenerationEscalationError) {
          // Escalar a revisión manual
          scene.status = 'manual_review_required';
          await this.notifyUser(error.context);
          return this.waitForManualFix(scene);
        }
        throw error;
      }
    });

    const completedScenes = await Promise.all(scenePromises);

    // Approval gate: Usuario revisa clips generados
    await this.waitForUserApproval('clips', state.videoId);

    // Layer 4: Composición final
    const composition = await this.agents.composition.composeFinalVideo(state.videoId);

    return {
      videoId: state.videoId,
      concept,
      scenes: completedScenes,
      finalVideo: composition.finalPath,
    };
  }

  // Manejo de feedback: Usuario puede iterar cualquier capa
  async handleIterationRequest(videoId: string, layer: Layer, feedback: string) {
    const video = await this.videoService.getById(videoId);

    switch (layer) {
      case 'concept':
        // Re-generar concepto completo
        return this.agents.concept.generateConcept(video.projectId, feedback);

      case 'prompt':
        // Re-generar prompts para todas las escenas
        const scenes = await this.sceneService.getByVideoId(videoId);
        return Promise.all(scenes.map(s =>
          this.agents.prompt.craftMiniMaxPrompt(s, feedback)
        ));

      case 'generation':
        // Re-generar clips específicos
        return this.regenerateClips(videoId, feedback);

      case 'composition':
        // Re-componer con ajustes
        return this.agents.composition.composeFinalVideo(videoId);
    }
  }
}

Estado & Eventos del Workflow

State Machine & Event Bus
workflow-state.ts 
// Estado de cada capa (layer)
interface LayerState {
  status: 'pending' | 'running' | 'completed' | 'failed' | 'manual_review';
  data?: T;
  error?: string;
  attempts: number;
  startedAt?: Date;
  completedAt?: Date;
}

// Estado completo del workflow
interface WorkflowState {
  videoId: string;
  status: WorkflowStatus;
  layers: {
    concept: LayerState;
    prompts: LayerState;
    generation: LayerState;
    composition: LayerState;
  };
  // Permite saltar a cualquier capa para iterar
  currentFocus?: 'concept' | 'prompts' | 'generation' | 'composition';
}

// Eventos para UI en tiempo real
type WorkflowEvent =
  | { type: 'layer_started'; layer: string; timestamp: number }
  | { type: 'layer_completed'; layer: string; result: unknown }
  | { type: 'layer_failed'; layer: string; error: string; canRetry: boolean }
  | { type: 'approval_required'; layer: string; preview: unknown }
  | { type: 'iteration_requested'; fromLayer: string; toLayer: string; feedback: string }
  | { type: 'auto_retry_triggered'; layer: string; attempt: number; reason: string };

// EventBus para comunicación entre agentes y UI
class WorkflowEventBus {
  private listeners = new Map void>>();

  emit(event: WorkflowEvent) {
    const layerListeners = this.listeners.get(event.type);
    layerListeners?.forEach(cb => cb(event));

    // Broadcast a listeners globales
    this.listeners.get('*')?.forEach(cb => cb(event));
  }

  on(eventType: string, callback: (event: WorkflowEvent) => void) {
    if (!this.listeners.has(eventType)) {
      this.listeners.set(eventType, new Set());
    }
    this.listeners.get(eventType)!.add(callback);
  }
}

// Ejemplo: UI escucha eventos
const eventBus = new WorkflowEventBus();

eventBus.on('layer_started', (e) => {
  console.log(`🟡 ${e.layer} started`);
  updateUIStatus(e.layer, 'running');
});

eventBus.on('layer_failed', (e) => {
  console.log(`🔴 ${e.layer} failed: ${e.error}`);
  if (e.canRetry) {
    showRetryButton(e.layer);
  } else {
    showManualReviewModal(e.layer, e.error);
  }
});

eventBus.on('approval_required', (e) => {
  console.log(`🔵 Approval needed for ${e.layer}`);
  showApprovalGate(e.layer, e.preview);
});

Flujo Agentico con Feedback Loops

Tipos de Feedback

Regeneración

Si MiniMax falla o el clip es de baja calidad, el agente vuelve a Layer 2 para reescribir el prompt.

Iteración creativa

El usuario puede solicitar cambios desde cualquier punto, saltando a la capa correspondiente.

Approval Gates

Puntos donde el usuario aprueba/rechaza antes de continuar al siguiente layer.

Ejemplo: Iteración en Tiempo Real

1

Usuario: "Quiero un video sobre promoción de verano"

→ Layer 1 (Concepto) genera:

{ title: "Verano Fitness", scenes: 3, ... }
2

Usuario: "Cambia la primera escena, quiero que sea más dramática"

→ Salta a Layer 2 (Prompt), modifica:

{ emotion: "dramatic", lighting: "chiaroscuro", ... }
3

Layer 3: Genera el video con MiniMax...

→ Quality gate detecta: el video está borroso

Auto-retry: Consulta a Layer 2 → Agrega "sharp focus, 4k quality" al prompt → Re-genera

4

Usuario: "Perfecto, pero agrega el logo más grande"

→ Layer 4 ajusta composición:

{ overlay: { logo: { scale: 1.5, position: "top-center" } } }

Implicaciones para la UI

Vista por Capas

La UI debe permitir "bajar" a cualquier capa y editar. No es un wizard lineal.

Estado en Tiempo Real

Cada capa muestra su estado: "pensando", "generando", "revisar", "aprobado".

Versionamiento

Cada iteración crea una rama. El usuario puede comparar y volver a versiones anteriores.
