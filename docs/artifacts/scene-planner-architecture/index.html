🏗️ Jerarquía de Contextos

LEVEL 1 ProjectContext brand · audience · guidelines
LEVEL 2 VideoContext theme · objective · visualStyle · settings
{
  theme: "GymSpace - Transformación Fitness",
  objective: "conversion",
  visualStyle: { mood: "energetic", pacing: "fast" },
  settings: { duration: 60, aspectRatio: "9:16" }
}
LEVEL 3 SceneContext action · emotion · visualElements · textOverlay
{
  action: "Hook: Captura atención con movimiento dinámico",
  emotion: "excitement",
  visualElements: { people: 1, location: "gym", lighting: "studio" },
  textOverlay: { headline: "¿Listo para transformar tu vida?", position: "center" }
}

✍️ Construcción de Prompts

buildScenePrompt() — ScenePlanner 
// Paso 1: Ação principal
if (sceneContext.action) {
  parts.push(sceneContext.action);
}

// Paso 2: Elementos visuais
if (sceneContext.visualElements) {
  const ve = sceneContext.visualElements;
  if (ve.people) parts.push(`${ve.people} person${ve.people > 1 ? 's' : ''}`);
  if (ve.location) parts.push(`at ${ve.location}`);
  if (ve.lighting) parts.push(`${ve.lighting} lighting`);
}

// Paso 3: Estilo do video context
if (videoContext.visualStyle?.mood) {
  parts.push(`${videoContext.visualStyle.mood} mood`);
}

// Resultado: prompt final
return parts.join(', ');
Prompt generado — escena 1 (Hook)
Hook: Capture attention with dynamic movement, 1 person at gym studio lighting, energetic mood
#gym #fitness #workout #motivation

📦 ScenePlan — Salida del Planner

api/core/planning/types.ts — ScenePlan interface 
interface ScenePlan {
  sequence: number;        // 1, 2, 3...
  description: string;      // "Hook: Captura atención"
  duration: number;        // segundos (ej: 6)
  prompt: string;         // prompt para MiniMax
  context: SceneContext;   // contexto detallado
}

interface PlanningResult {
  title: string;
  description: string;
  totalDuration: number;
  scenes: ScenePlan[];
}
